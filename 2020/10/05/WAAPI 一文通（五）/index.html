项目分析

功能简述

从项目名 WAAPI Database 中不难看出，本程序的主要目的是对 Wwise 中各种信息进行抓取，其核心功能是通过 ak.wwise.core.object.get 来实现的。
抓取后的数据会通过 SQLite 保存在本机数据库之中，这是一款轻量且免配置的数据库工具。如果读者不熟悉数据库也没关系，可把数据库当做是一种结构更复杂但更易于管理的电子表格结构，暂时可不予深究。
WAAPI Database 的运行方式是借助 Wwise 在 SoundBank 生成时的 PostXXXX，通过这个功能可在生成 SoundBank 后执行一个 Command Line Tools 文件，从而触发 WAAPI Database 中的功能代码。
