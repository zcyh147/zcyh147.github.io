公众号 FAQ

溪夜
2020-09-03
置顶

大家好，我是溪夜。
这里汇总了有关本号的常见疑问与其解答，希望对大家有所帮助。

本文目录：
[toc]

关于文章更新频次

我认为持续保持更新的技术文章，才会体现出作者对这篇作品的重视和对分享的诚意。

基于此概念，本公众号内所有文章都会不定时升级，更新取决于文章内容的增减、是否存在 Bug 需要修改等。人无完人，找到并改正错误是一件美妙的事。对我来说只有下方各位真挚的留言有收藏意义，而点击、在看、点赞数这类文字游戏都如过眼云烟。
分享有价值的知识、实现技术开源才是我的初心。
当诸位发现某些老文突然冠以“第 N 版”诈尸出现时请勿奇怪，可打开文章看看开头这次的更新是什么。

关于公众号文章的组织结构

有结构的知识分享无比重要。
从目前公众号所能用的文章结构组织方法考虑，建议诸位通过主菜单的“历史文章”按钮查看经我组织后的文章结构。

关于提问

鉴于本人精力有限，留言及私信中关于技术类的提问着实无法保证会一一回复，还请谅解。
如果读者诸君有问题需要解答，可尝试按以下步骤解决：

  1. 使用搜索引擎（如 Google）进行中英文双语检索，查看是否存在可用解决方案。
  2. 在知乎搜索溪夜（头像为肖斯塔科维奇），通过值乎进行提问。

更优的阅读方案

涉及代码类的文章我可能在 GitHub 上也建立一份代码仓库，以提供更优秀的阅读和传播方案。
本人的个人博客比较混乱，待完善完毕后也会向诸位开放。

关于转载

如需转载，请对公众号直接发消息，我看到后会给与回复。

关于打赏

人生苦短，世事无常，能通过技术分享的文字与诸君相识是一种缘分。感谢这个时代，让渺小的个体也有机会通过文字记录自己的研究成果。我把打赏金额默认高于50的额度全部去掉了，技术分享遵循开源精神，打赏只是鼓励，有无均可。
若是诸位看到对自己有帮助的文章愿意打赏请我喝杯咖啡，从瓶装咖啡（我蛮喜欢 Costa 的瓶装咖啡的hhh性价比高）到星巴克我都不会介意。：）
能够分享对大家有用的研究我很开心，您的阅读和收获才是对我最大的鼓励。
感谢诸位。
