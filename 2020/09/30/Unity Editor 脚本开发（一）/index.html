技术音频的 Unity 编辑器工具开发实践（一）概述
2020-09-30
字数总计: 430|阅读时长: 1 分钟

大家好，我是溪夜。
对技术音频设计师而言，开发新工具无疑是最基本且重要的一项能力。
在之前的文章中，我们已经聊过使用 PySimpleGUI 为 Python 脚本快速添 GUI 的详细方法。PySimpleGUI 系列文章所讨论的工具开发，是独立于游戏和声音引擎之外的工具。
而在这一系列文章中，会分享通过 Unity 的 Editor Scripting 在游戏引擎中开发自定义工具的实践方法。
与纯引擎程序员的开发方向不同，技术音频设计师一般不会在 Unity Editor 中创建关卡快速构建器这样的工具。改善并创建 Unity 与音频中间件集成时的有关工具，才会是我们主要的学习方向。
作为偏实战的技术分享，在这系列文章结束后你会完成几个简单和一个稍复杂的 Unity 技术音频类工具的制作。

本系列文章阅读需要的前置知识：

  • 了解游戏引擎使用、 Wwise Unity 整合和 C# 基本语法等必备知识。
  • 对 WAAPI 比较熟悉，了解在 C# 下的 WAAPI 调用。
